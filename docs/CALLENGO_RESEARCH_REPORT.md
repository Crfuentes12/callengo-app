# 1. RESUMEN EJECUTIVO

## 1.1 Qué es Callengo

**Callengo es una plataforma de llamadas telefónicas automatizadas impulsada por Inteligencia Artificial** que permite a las empresas realizar campañas de llamadas a escala, verificar información de contactos, calificar leads y confirmar citas - todo de forma automática y sin intervención humana.

### Definición en una Frase
> "Callengo es tu equipo de llamadas que trabaja 24/7, nunca se cansa, siempre sigue el guión perfecto y te entrega resultados medibles."

### Mercado Objetivo
- **Empresas B2B** con bases de datos de contactos
- **Equipos de ventas** que pierden tiempo en llamadas repetitivas
- **Clínicas y servicios** con altas tasas de no-shows
- **Empresas de servicios** que necesitan verificar información constantemente
- **Startups y PYMEs** que no pueden permitirse un call center

# 2. EL PROBLEMA QUE RESUELVE CALLENGO

## 2.1 Los Dolores del Usuario (Pain Points)

### DOLOR #1: Bases de Datos Sucias
**"Emails que rebotan, teléfonos que no existen, leads duplicados"**

Las empresas gastan miles de dólares en:
- Campañas de email marketing que llegan a direcciones inválidas
- SDRs llamando a números desconectados
- CRMs inflados con data basura
- Decisiones de negocio basadas en métricas falsas

**Impacto Real:**
- 25-30% de los datos de contacto se desactualizan cada año
- El costo promedio de un lead malo es de $100-500
- Los equipos de ventas pierden 40% de su tiempo en tareas administrativas

### DOLOR #2: No-Shows y Citas Perdidas
**"Cada cita que se pierde es dinero tirado"**

Los no-shows afectan especialmente a:
- Clínicas médicas y dentales
- Consultorías y servicios profesionales
- Talleres y servicios automotrices
- Salones de belleza y spas
- Restaurantes con reservaciones

**Impacto Real:**
- Tasa promedio de no-shows: 15-30%
- Costo promedio por no-show: $50-200
- Pérdida anual para una clínica pequeña: $20,000-100,000

### DOLOR #3: SDRs Quemados con Leads Basura
**"Tu equipo de ventas perdiendo tiempo en conversaciones que no van a ningún lado"**

Los equipos de ventas enfrentan:
- Leads no calificados que consumen tiempo valioso
- Frustración y alta rotación del personal
- Métricas de conversión artificialmente bajas
- Costos de adquisición de cliente inflados

**Impacto Real:**
- Un SDR promedio gasta solo 35% de su tiempo vendiendo
- Costo anual de un SDR: $50,000-80,000 (salario + beneficios + tools)
- Tasa de rotación promedio: 35% anual

# 4. LOS TRES AGENTES ESTRELLA

Callengo ofrece tres agentes de IA especializados, cada uno diseñado para resolver un problema específico del negocio.

---

## 4.1 AGENTE DE VALIDACIÓN DE DATOS
### "El Limpiador de Bases de Datos"

**Slug interno:** `data-validation`

### Problema que Resuelve
> "Deja de desperdiciar dinero en data mala. Emails que rebotan, teléfonos equivocados, leads duplicados."

### Qué Hace Este Agente
1. **Llama al contacto** para verificar que el número es correcto
2. **Confirma la identidad** de la persona o empresa
3. **Verifica/actualiza** información clave:
   - Email actualizado
   - Dirección física
   - Nombre del contacto correcto
   - Si el negocio sigue operando
4. **Marca el registro** con el estado de verificación
5. **Extrae nueva información** mencionada en la llamada

### Flujo de Conversación Típico
```
AGENTE: "Hola, mi nombre es Maya de [Tu Empresa].
        Estoy actualizando nuestra información de contacto.
        ¿Podría confirmar si esta es la empresa XYZ?"

CONTACTO: "Sí, somos XYZ"

AGENTE: "Perfecto. Tengo registrada la dirección en 123 Main St.
        ¿Sigue siendo correcta?"

CONTACTO: "No, nos mudamos. Ahora estamos en 456 Oak Ave."

AGENTE: "Entendido, 456 Oak Ave. ¿Y el mejor email para contactarlos?"

CONTACTO: "contacto@xyz.com"

AGENTE: "Muchas gracias por su tiempo. Que tenga un excelente día."
```

### Estados de Contacto que Genera
| Estado | Significado |
|--------|-------------|
| `Fully Verified` | Toda la información confirmada |
| `Research Needed` | Información parcial, requiere seguimiento |
| `No Answer` | No contestaron después de intentos |
| `Wrong Number` | Número incorrecto |
| `Number Disconnected` | Número fuera de servicio |
| `Voicemail Left` | Se dejó mensaje de voz |
| `For Callback` | Solicitaron llamar en otro momento |

### Resultados del Análisis
Después de cada llamada, la IA extrae:
```json
{
  "verifiedAddress": "456 Oak Ave, City, State 12345",
  "contactName": "John Smith",
  "verifiedEmail": "john@company.com",
  "businessConfirmed": true,
  "outcomeNotes": "Business moved, new address provided",
  "callSentiment": "positive",
  "customerInterestLevel": "medium"
}
```

### Casos de Uso Ideales
- Limpieza de CRM antes de campaña de marketing
- Actualización anual de base de datos
- Verificación de leads comprados
- Preparación para auditorías

## 4.2 AGENTE DE CONFIRMACIÓN DE CITAS
### "El Reductor de No-Shows"

**Slug interno:** `appointment-confirmation`

### Problema que Resuelve
> "Cada cita perdida es dinero que se va. Equipos esperando a clientes que nunca llegan."

### Qué Hace Este Agente
1. **Llama 24-48 horas antes** de la cita programada
2. **Confirma asistencia** del cliente
3. **Ofrece reagendar** si el cliente no puede
4. **Recopila información adicional** necesaria para la cita
5. **Envía recordatorio** y confirmación

### Flujo de Conversación Típico
```
AGENTE: "Hola, le llamo de [Clínica/Empresa] para confirmar
        su cita del día jueves a las 3pm con el Dr. García.
        ¿Nos confirma su asistencia?"

CLIENTE: "Ah sí, pero creo que no voy a poder ir a esa hora."

AGENTE: "Entiendo, no hay problema. ¿Le gustaría que
        reprogramemos para otro horario? Tenemos disponibilidad
        el viernes por la mañana."

CLIENTE: "Viernes a las 10am me queda bien."

AGENTE: "Perfecto, queda agendado para el viernes a las 10am.
        ¿Hay algo que debamos preparar para su visita?"

CLIENTE: "Sí, voy por un chequeo de rutina."

AGENTE: "Anotado. Le esperamos el viernes. ¡Que tenga un buen día!"
```

### Acciones Automáticas
- **Si confirma:** Marca como confirmada, envía recordatorio
- **Si cancela:** Ofrece nueva fecha, libera el slot
- **Si no contesta:** Programa reintento, deja voicemail
- **Si pide reagendar:** Ofrece alternativas disponibles

### Métricas de Éxito
| Métrica | Sin Callengo | Con Callengo |
|---------|--------------|--------------|
| Tasa de no-shows | 25-30% | 8-12% |
| Citas confirmadas | 40% | 85%+ |
| Tiempo administrativo | 2-3 hrs/día | 0 hrs |

### Casos de Uso Ideales
- Clínicas médicas y dentales
- Servicios profesionales (abogados, contadores)
- Salones de belleza y spas
- Talleres mecánicos
- Cualquier negocio con citas

## 4.3 AGENTE DE CALIFICACIÓN DE LEADS
### "El Filtro Inteligente"

**Slug interno:** `lead-qualification`

### Problema que Resuelve
> "Tu equipo de ventas pierde tiempo en leads basura. Filtra automáticamente y salva su energía."

### Qué Hace Este Agente
1. **Contacta leads entrantes** inmediatamente
2. **Califica según criterios** definidos (BANT, MEDDIC, etc.)
3. **Determina nivel de interés** y urgencia
4. **Recopila información adicional** para ventas
5. **Prioriza y routea** los leads calificados

### Criterios de Calificación
El agente evalúa:
- **Budget:** ¿Tienen presupuesto?
- **Authority:** ¿Es el decisor?
- **Need:** ¿Tienen una necesidad real?
- **Timeline:** ¿Cuándo planean decidir?
- **Fit:** ¿Es nuestro cliente ideal?

### Flujo de Conversación Típico
```
AGENTE: "Hola, mi nombre es Ryan de [Empresa]. Vi que
        descargaste nuestra guía sobre [Tema]. ¿Tienes
        unos minutos para contarte cómo podemos ayudarte?"

LEAD: "Sí, estamos buscando una solución para..."

AGENTE: "Interesante. ¿Cuántas personas hay en tu equipo
        que usarían la herramienta?"

LEAD: "Somos unos 15."

AGENTE: "Perfecto. ¿Y ya tienen asignado presupuesto
        para este tipo de solución?"

LEAD: "Sí, tenemos aprobado hasta $X por mes."

AGENTE: "Genial. ¿Quién más en tu empresa participa
        en este tipo de decisiones?"

LEAD: "Mi jefe, el director de operaciones."

AGENTE: "Excelente. Basado en lo que me cuentas, creo
        que podemos ayudarles mucho. ¿Te parece si uno
        de nuestros especialistas te llama mañana a
        esta misma hora para una demo personalizada?"
```

### Niveles de Interés que Detecta
| Nivel | Descripción | Acción Sugerida |
|-------|-------------|-----------------|
| `high` | Listo para comprar | Llamada de ventas inmediata |
| `medium` | Interesado pero no urgente | Nurturing + seguimiento |
| `low` | Curiosidad sin intención | Email marketing |
| `none` | Sin interés | No contactar |

### Información que Extrae
```json
{
  "extractedData": {
    "companySize": "15 employees",
    "budget": "$X per month approved",
    "decisionMaker": "Director of Operations",
    "timeline": "Within this quarter",
    "currentSolution": "Using competitor X"
  },
  "outcome": "Qualified lead - ready for sales call",
  "nextActions": [
    "Schedule demo call for tomorrow",
    "Send case study from similar company"
  ],
  "callQuality": {
    "rating": 9,
    "reason": "Engaged prospect, clear need, budget confirmed"
  }
}
```

### Casos de Uso Ideales
- Follow-up de leads de marketing
- Calificación de formularios web
- Speed-to-lead (contacto inmediato)
- Re-engagement de leads fríos

|-----------|---------------|

| **Manejo de interrupciones** | Sí, natural |
| **Idiomas soportados** | 10+ |
| **Llamadas concurrentes** | Según plan |

### Parámetros de Llamada Configurables
```javascript
{
  voice: "maya",              // Voz del agente
  wait_for_greeting: true,    // Esperar que contesten
  record: true,               // Grabar la llamada
  max_duration: 5,            // Duración máxima (minutos)
  voicemail_action: "leave_message",  // Qué hacer si voicemail
  answered_by_enabled: true,  // Detectar humano vs máquina
  model: "enhanced",          // Modelo de IA
  language: "en",             // Idioma
  temperature: 0.7,           // Creatividad de respuestas
  background_track: "office"  // Sonido de fondo de oficina
}
```

## 5.2 Motor de Análisis (OpenAI GPT-4o)

Cada llamada es analizada automáticamente usando **GPT-4o**, el modelo más avanzado de OpenAI.

### Información Extraída Automáticamente
1. **Datos estructurados** (email, teléfono, dirección)
2. **Campos validados** (qué se confirmó vs. qué cambió)
3. **Nueva información** (datos que no teníamos)
4. **Resultado de la llamada** (outcome)
5. **Próximos pasos** (acciones recomendadas)
6. **Calidad de la llamada** (rating 1-10)
7. **Sentimiento** (positivo, neutral, negativo)
8. **Nivel de interés** (alto, medio, bajo, ninguno)

### Ejemplo de Análisis Real
```json
{
  "extractedData": {
    "contactName": "María García",
    "email": "maria@empresa.com",
    "role": "Gerente de Operaciones"
  },
  "validatedFields": {
    "phone": "Confirmed",
    "address": "Updated: 456 New Street"
  },
  "newInformation": {
    "preferredContactTime": "mornings",
    "budgetRange": "$5,000-10,000/month"
  },
  "outcome": "Qualified lead, interested in demo",
  "nextActions": [
    "Send calendar invite for Thursday 10am",
    "Prepare custom proposal with volume pricing"
  ],
  "callQuality": {
    "rating": 8,
    "reason": "Engaged conversation, clear interest shown"
  }
}
```

## 5.3 Detección y Manejo de Voicemail

### Flujo de Voicemail
1. **Detección automática** - Identifica cuando salta buzón
2. **Mensaje personalizable** - Deja mensaje configurado
3. **Follow-up automático** - Programa reintento
4. **Tracking** - Registra que se dejó mensaje

### Configuración de Voicemail
```javascript
{
  voicemail_enabled: true,
  voicemail_detection_enabled: true,
  voicemail_message: "Hola, le llamo de [Empresa]. Por favor...",
  voicemail_action: "leave_message" // o "hangup"
}
```

---

## 5.4 Sistema de Follow-ups Automáticos

### Condiciones para Follow-up
El sistema puede programar seguimientos automáticos cuando:
- No contestaron la llamada
- Se dejó voicemail
- Solicitaron llamar después
- La llamada se cortó
- Se necesita más información

### Configuración de Campañas con Follow-up
```javascript
{
  follow_up_enabled: true,
  follow_up_max_attempts: 3,        // Máximo de reintentos
  follow_up_interval_hours: 24,     // Horas entre intentos
  follow_up_conditions: {
    "no_answer": true,
    "voicemail": true,
    "callback_requested": true
  }
}
```

## 7.2 Modelo de Ingresos

### Fuentes de Ingreso

1. **Suscripciones mensuales/anuales**
   - Ingreso recurrente predecible
   - Descuento del ~12% en planes anuales

2. **Minutos de overage**
   - Cobro por minuto adicional sobre el plan
   - Tasas decrecientes según plan ($0.80 → $0.20/min)

3. **Usuarios adicionales** (Plan Teams)
   - $79/mes por usuario adicional

4. **Enterprise deals**
   - Contratos anuales personalizados
   - Volume discounts

### Economía del Modelo

```
Ejemplo: Cliente en Plan Business

Minutos incluidos: 1,000
Precio base: $299/mes
Costo Callengo por minuto: ~$0.12 (Bland AI cost)
Margen bruto base: ~60%

Si usa 1,500 minutos:
- 1,000 incluidos en plan
- 500 overage × $0.35 = $175 extra
- Ingreso total: $474
- Margen ~65%
```
